Most recent work

SmartDiet-Planner

SmartDiet-Planner is a web-based project that utilizes the Edamam Nutrition Analysis API to create dynamic, personalized diet plans for users. It aims to enhance nutritional awareness by providing real-time insights into the calorie count, macronutrients, and overall health score of meals. By allowing users to input their food preferences or meals, the system analyzes ingredients and returns detailed nutritional breakdowns. The project helps users make informed dietary choices, promoting a healthier lifestyle through technology-driven meal tracking and optimization.

EventSphere-System

EventSphere-System is a project focused on streamlining event planning and coordination through a centralized, user-friendly web platform. It empowers organizers to efficiently manage event creation, participant registration, and real-time updates with ease. Built using modern web technologies and a robust backend, the system simplifies complex workflows involved in event logistics. By integrating responsive design and database-driven automation, EventSphere enhances the overall event experience for both hosts and attendees, making planning seamless, organized, and effective.
